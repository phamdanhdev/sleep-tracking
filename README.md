
# Sleep Tracking Project

This is a Sleep Tracking Web App use for tracking sleep quality.


## Tech Stack

**Client:** ReactJS, AntDesign, Redux, Redux-Thunk, JWT, Google API

**Server:** NodeJS, express, mongoose
